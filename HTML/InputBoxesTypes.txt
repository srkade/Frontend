Progress and Meter Tag
-->Meter tag is used to measure anything in Quantitively in html
ex-->
<meter min="0" ,max="100", low="40",optimum="60" high="80" ,value="85">
</meter>

min:- min is used to indicate minimum value

max:- max  is used to indicate maximum value

value:-It is used to provide the current value for the input 
type or any input boxes

high:-it is used to indicate the higher value in the meter tag

low:-it is used to indicate the lower value in the meter tag

optimum:-It is used to indicate average value in meter tag
